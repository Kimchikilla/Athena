# Athena

**한 줄 설명: AI한테 가짜 한국인 5천 명을 만들어 투표시키고, 누가 이길지 봐보는 실험.**

2026년 6월 3일 한국 지방선거 시·도지사 선거가 대상이고, 결과는 진짜 예측이 아니라 *AI가 인구통계를 어떻게 정치적으로 해석하는지* 보는 거예요. (왜 진짜 예측이 아닌지는 [아래](#그래서-이게-진짜-예측이냐)에서)

---

## 이게 뭘 하는 건데?

```
┌──────────────────────────────────────────────────────────────┐
│ 1단계  중앙선거관리위원회 사이트에서 후보자 명단 긁어오기      │
│        → 17개 시·도 × 6개 선거 × 약 8,300명 후보              │
│                                                              │
│ 2단계  엔비디아가 만든 "가짜 한국인 100만 명" 데이터셋에서    │
│        시·도별 300명씩 = 5,100명 뽑기                         │
│        (예: 대구 47세 남성, 회계사무원, 아파트 거주, 자녀 둘) │
│                                                              │
│ 3단계  로컬 PC에서 AI(Gemma 4)에게 부탁하기:                 │
│        "이 사람이 되어서 후보 6명 중 한 명 골라봐"            │
│                                                              │
│ 4단계  4,800번 반복하고 표 집계                              │
└──────────────────────────────────────────────────────────────┘
```

## 결과 살짝 보기 (대구시장)

| 후보 | 시뮬레이션 득표율 |
|---|---|
| 김한구 (무소속) | **90.5%** |
| 이재만 (국민의힘) | 8.2% |
| 김부겸 (더불어민주당) | **1.4%** |
| 홍석준·이진숙·이수찬 | 0% |

뭔가 이상하죠? **김부겸이 1.4%** 라는 건 현실 정치를 안다면 말이 안 됩니다. 김부겸은 전 국무총리에 4선 의원이고 대구 출신이에요. 실제론 강력한 후보입니다.

→ 이게 바로 **AI가 진짜 표심을 모른다는 증거**입니다. AI는 페르소나 텍스트에 "공장 노동자"가 많이 나오니까 무소속 노동자 출신 후보(김한구)에게 표를 몰아준 거예요.

전체 16개 시·도 결과는 [`data/vote_summary.csv`](data/vote_summary.csv) 참고.

## 그래서 이게 진짜 예측이냐?

**아닙니다.** 이번 실험에서 명확하게 드러난 AI의 4가지 함정:

1. **현직 무조건 1등** — 강원 김진태 100%, 경북 이철우 99%. AI가 "현직"이라는 단어만 보면 그쪽으로 거의 다 몰림
2. **직업 라벨 단순 매칭** — 페르소나에 "노동자" 많으면 노동자 후보로 우르르
3. **정보 없으면 기권** — 후보 경력이 짧게 나오면 충북 73%, 인천 93%가 "기권" 선택
4. **정당지지 데이터 없음** — 데이터셋에 "이 사람 보수냐 진보냐" 같은 라벨이 없어서 AI가 직업/거주지만 보고 추측

요약: **AI는 실제 한국 유권자가 아닙니다.** 그냥 LLM이 인구통계를 정치 선택으로 어떻게 변환하는지 보여주는 실험이에요.

## 직접 돌려보고 싶다면

### 필요한 것

- **PC**: GPU 8GB VRAM 이상 (RTX 4060/5060 정도)
- **디스크 여유**: 15GB
- **시간**: 약 4시간 (시뮬 3시간 + 다운로드/세팅 1시간)
- **OS**: Windows 11 (다른 OS도 가능할 듯)

### 5단계

```bash
# 1. 환경
python -m venv .venv
.venv/Scripts/python.exe -m pip install -r requirements.txt
winget install --id=Ollama.Ollama --silent
ollama pull gemma4:e4b

# 2. 후보 명단 긁기 (5분)
.venv/Scripts/python.exe -m nec.cascade
.venv/Scripts/python.exe -m nec.crawler

# 3. 가짜 한국인 5,100명 샘플링 (10분)
.venv/Scripts/python.exe -m sim.sample_personas

# 4. 투표 시뮬레이션 (3시간)
.venv/Scripts/python.exe -m sim.runner

# 5. 결과 표로 정리
.venv/Scripts/python.exe -m sim.aggregate
```

중간에 멈춰도 다시 돌리면 끊긴 곳부터 이어집니다.
